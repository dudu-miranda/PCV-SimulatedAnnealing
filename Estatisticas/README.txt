Para todas as combinações foram utilizadas as seguintes configurações:
Temperatura Inicial = 100
SAMAX = 10
Alpha = 0.9
Reseta Inicial = 1
As alteracoes que houveram foram as do metodo inicial escolhido, do numero de reaquecimentos e do metodo de gerar vizinhos.

Metodo inicial = Método para geração da primeira solução.
Numero de reaquecimentos = Numero de vezes que foi reaquecida a temperatura do Simulated Annealing.
Metodo vizinho = Método de gerar vizinhos que foi utilizado, swap ou guloso.
OBS: O metodo swap está escrito como swap____, somente para que fique organizado o arquivo de saída.
Media = Media das 100 iteracoes por combinacao de entradas.
Melhor encontrado = Melhor solucao encontrada na combinacao de entradas atual.
Coeficiente de variacao = Melhor encontrado/Melhor Solucao
